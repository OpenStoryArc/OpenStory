Subject Hierarchy: Before → After

Current: Flat
Proposed: Hierarchical

Subscription Patterns

NATS Wildcard Subscriptions
events.openstory.06907d46.>
→ Main agent + ALL 5 subagents (full session visibility)
events.openstory.06907d46
→ Main agent only (no subagents)
events.openstory.06907d46.agent.>
→ Only subagents (5 agents, not the main)
events.openstory.06907d46.agent.a6dcf911
→ One specific subagent ("Explore PatternView types and data")
events.openstory.>
→ All sessions in the openstory project
events.>
→ Everything (all projects, all sessions)
Improvement
Parent-child relationships are structural. No payload inspection needed. The subject IS the routing key.

Event Flow

Sources

File Watcher / Hooks
translate_line(raw) → CloudEvent

Subject composition:
  project = resolve_project(cwd)
  session = raw.sessionId
  agent   = raw.agentId   // null for main

  if agent:
    "events.{project}.{session}.agent.{agent}"
  else:
    "events.{project}.{session}"
Improvement: session_id is real
No more synthesized "agent-{id}" session IDs.
Identity is the composite: (session_id, agent_id)
The subject encodes both.

Transport

Stream: eventsevents.>
Single durable stream captures all subjects.
Consumers filter by pattern.
Stream: patternsNEW
Detected patterns published back to bus.
patterns.{project}.{session}
Consumers can react to patterns independently.
Stream: changeschanges.>
Broadcast messages for distributed UI clients.

Consumers (Independent)

persist
filter: events.>
CloudEvent → SQLite + JSONL (no transform)
patterns
filter: events.>
CloudEvent → EvalApply → Sentence → PatternEvent
→ publish to patterns.{project}.{session}
projections
filter: events.>
CloudEvent → SessionProjection (tokens, files, duration)
broadcast
filter: events.> + patterns.>
CloudEvent → ViewRecord → WireRecord → WebSocket
Improvement: Independent consumers
Each consumer has its own failure domain.
Pattern detection can lag without blocking persistence.
New consumers (analytics, alerts) add without changing existing code.

Type at Each Boundary

Source          JSONL line (serde_json::Value)
    │               raw.sessionId, raw.agentId, raw.message
    │ translate_line()
    ▼
CloudEvent      The canonical type on the bus
    │               id, type, subtype, time, agent
    │               data.raw (immutable), data.agent_payload (typed lift)
    │               data.agent_payload.tool_outcome (domain event)
    │
    │ publish to events.{project}.{session}[.agent.{agent_id}]
    ▼
NATS            JSON-serialized IngestBatch { session_id, project_id, events }
    │
    │ consumer deserializes
    ▼
CloudEvent      Same type, no transformation during transport
    │
    ├──▶ persist:     CloudEvent → SQLite row (no transform)
    │
    ├──▶ patterns:    CloudEvent
    │                     → step(accumulator, event) → StructuralTurn
    │                     → SentenceDetector → PatternEvent
    │                         type: "turn.sentence"
    │                         session_id, event_ids, metadata
    │                         metadata.applies[].tool_outcome.agent_id  ← THE LINK
    │
    ├──▶ projections: CloudEvent → SessionProjection update
    │
    └──▶ broadcast:   CloudEvent → ViewRecord → WireRecord → WebSocket
