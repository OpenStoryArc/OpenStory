The Fix: Decomposing Translator

The API returns content blocks separately. Pi-mono bundles them. The translator un-bundles them. One JSONL line → N CloudEvents (one per content block).

Decomposed: 1 line → 3 CloudEvents
✓ message.assistant.thinking
✓ message.assistant.text — "Let me read the file first..."
✓ message.assistant.tool_use — read("/tmp/test-broken.py")

All content preserved. Same structure the API returned. Nothing invisible.
